1. Создать репозиторий git init
1. Создать файл .editorconfig
.editorconfig нужен для того, чтобы у редакторов кода были одинаковые настройки
можно установить плагины editorconfig для редакторов кода
то как сохраняется файл в при сохранении
2. создать .gitignore
3. npm init -y создать файл package.json
4. поставить gulp локально npm install --save-dev gulp
если нам нужен какой-то код в самом проекте когда сайт мы открываем и это все запускается и чтобы именно это работало
уже в самом сайте например jquery тогда мы ставим в dependencies, если нам нужно на этапе разработки, тогда ставим в
devDependencies
5. создать gulfile.babel.js для этого нужно поставить babel-register также нужно установить сам babel потому что babel-register -
это всего лишь надстройка для babel нам нужно поставить babelcore - это сам babel
npm install @babel/core @babel/register --save-dev
6. так же нужно поставить какой-нибудь плагин либо пресет потому что из коробки babel - это компилятор и он ничего не делает
@babel/preset-env - собирает в себя все плагины и позволяет писать на последнем js
npm i babel/preset-env --save-dev
чтобы пресет заработал его нужно указать  конфигурационном файле
7. создать файл .babelrc - это конфигурационный файл для babel
